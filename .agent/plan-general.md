# Plan General y Roadmap Estratégico

Este documento consolida la visión macro operativa del proyecto, dirigida por Santi.

## Iniciativas Estratégicas

### 1. Manual de UI/Maquetación (Referente: Linear.app)
**Objetivo:** Desarrollar un manual visual base para el ERP tomando como referencia de diseño y maquetación a [linear.app](https://linear.app). 
**Enfoque:** Analizar y documentar cómo empaquetan y estructuran sus elementos visuales (layout, espacios, componentes).
**Nota:** Sin estructuras complejas, puramente un manual de componentes y diseño.

### 2. Desarrollo de Vistas de Frontend (Vue + Quasar)
**Objetivo:** Construir las interfaces visuales (vistas de usuario) utilizando **Vue.js y el framework Quasar**.
**Contexto:** Los datos y vistas SQL analíticas (`resumen_ventas_*`) **ya existen y están listas** en la base de datos integradora. El paso actual es puramente frontend: conectar y renderizar esos datos de forma corporativa basándonos en el manual de diseño del punto 1.

### 3. Asistente con Inteligencia Artificial (Bots de Telegram)
**Objetivo:** Implementar capacidades de IA en los bots de Telegram de la empresa.
**Propósito:** Que el equipo pueda interactuar ("hablar") mediante lenguaje natural directamente con los datos consolidados del ERP desde Telegram.
